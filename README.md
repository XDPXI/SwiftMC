## üöÄ Building from Source

### Requirements

- **Java 25 JDK**
- **Git**
- Supported OS: Windows / macOS / Linux / FreeBSD

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/XDPXI/SwiftMC.git
   cd SwiftMC
   ```
2. Build using Gradle (the compiled JAR will be located in build/libs):
   ```bash
   ./gradlew build
   ```
